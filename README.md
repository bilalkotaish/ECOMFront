# 🛍️ Billy E-Commerce

A modern full-stack e-commerce application built with the **MERN Stack**. The platform delivers a seamless shopping experience with secure authentication, product management, shopping cart, wishlist, checkout, and an intuitive admin dashboard.

## 🚀 Live Demo

**🌐 https://billyecommerces.vercel.app/**

## ✨ Features

* User Authentication (JWT)
* Google Sign-In
* Product Search & Filtering
* Shopping Cart & Wishlist
* Stripe Payment Integration
* Order Management
* Address Management
* User Profile
* Admin Dashboard
* Product & Category Management
* Responsive Design

## 🛠️ Tech Stack

**Frontend**

* React.js
* Tailwind CSS
* Material UI
* Axios

**Backend**

* Node.js
* Express.js
* MongoDB
* Mongoose
* JWT Authentication

**Services**

* Stripe
* Firebase Authentication
* ImageKit

## 📦 Installation

```bash
git clone https://github.com/your-username/your-repository.git

cd your-repository

npm install
```

Start the development server:

```bash
npm run dev
```
